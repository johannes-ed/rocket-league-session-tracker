# rocket-league-session-tracker
A small pyqt app that shows you current playing session stats, wins, losses etc. Data comes from the new local rocket league game stats API. The main.py is the file to run with required dependencies installed. I will try to package it more at some point.

To activate the datastream, you have to edit a specific game file as specified in the RL API documentation:
https://www.rocketleague.com/en/developer/stats-api
