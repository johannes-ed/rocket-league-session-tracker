# rocket-league-session-tracker
A small pyqt app that shows you current playing session stats, wins, losses etc. Data comes from the new local rocket league game stats API. 

To activate the datastream, you have to edit a specific game file as specified in the RL API documentation:
https://www.rocketleague.com/en/developer/stats-api


You can now download the first release here on Github, unzip the folder and run the exe file in it. Dependencies are inside along with. Packaged with pyinstaller.
(windows defender might warn because of unsigned exe)
